#### 📂 What you’ll find in my repositories:
On my GitHub account you can find several repositories that showcase the projects and skills I developed throughout the course. The main programming language I worked with was Python, and I created Jupyter Notebooks to document and explain my work. Each repository contains a README for guidance. Below is an overview of my repositories and their main purpose:
- **[WSAA-coursework](https://github.com/FatimaBOliveira/WSAA-coursework)**: This repository contains a project that demonstrates how to connect an HTML page to a MySQL database, with both frontend and backend development. It also includes assessments involving interactions with web pages;
- **[applied-statistics-assessment](https://github.com/FatimaBOliveira/applied-statistics-assessment)**: It includes problems solved with statistical tests using Python;
- **[computer-infrastructure-assessment](https://github.com/FatimaBOliveira/computer-infrastructure-assessment)**: This project demonstrates how a CLI can be used to interact with the system. It also uses GitHub to automate tasks, retrieve information and generate files;
- **[Programming-for-data-analytics](https://github.com/FatimaBOliveira/Programming-for-data-analytics)**: This project involves processing weather data from Met Éireann to analyse and assess whether the location studied is suitable for installing windmills for energy production. It also includes assessments where the programs read files to generate plots, and one that simulates a dice game using Python;
- **[My-Data-Analtyics-Project](https://github.com/FatimaBOliveira/My-Data-Analytics-Project)**: This project studies the characteristics of penguins and explores associations between variables using Python;
- **[pands-project](https://github.com/FatimaBOliveira/pands-project)**: A project based on the Iris dataset, making comparisons between species using Python;
- **[pands-weekly-tasks](https://github.com/FatimaBOliveira/pands-weekly-tasks)**: Solved weekly problems and explored different Python functions;
- **[pands-mywork](https://github.com/FatimaBOliveira/pands-mywork)**: It contains small tasks completed with Python during lectures.
